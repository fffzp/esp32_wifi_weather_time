# esp32_wifi_weather_time

#### 介绍
基于IDF VScode 和 LVGL 开发的Esp32s3天气闹钟功能
1.搜索附近wifi列表，连接成功后记录在nvs，下次搜索到相同wifi自动连接
2.获取网络时间，三天的天气信息，地点可自定义
3.闹钟时间到，音乐提示声
